## 🚀 Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Installation

```bash
# Clone repository
git clone https://github.com/maharshi027/portfolio.git

# Navigate to project
cd portfolio

# Install dependencies
npm install

# Start dev server
npm run dev
```

Visit `http://localhost:5173` to view the portfolio locally.
